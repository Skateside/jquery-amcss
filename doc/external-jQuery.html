(static) AM_PREFIX :String

Source:
  • jquery.amcss.js, line 441

The prefix for the AMCSS attributes. This intentionally includes the hyphen so that the value can be set to an empty string ("") if no prefix is wanted. Changing this value will create different results in different values returned from jQuery.normaliseAm.

Type:
  • String
Example

Changing $.AM_PREFIX updates $.normaliseAm

$.AM_PREFIX = "am-"; // This is the default.
$.normaliseAm("button"); // -> "am-button"

$.AM_PREFIX = "data-am-";
$.normaliseAm("button"); // -> "data-am-button"

(static) amHooks :Object.<AMCSS_hook>

Source:
  • jquery.amcss.js, line 485

A collection of hooks that change the behaviour of getting or setting AMCSS attributes. They are only called when using jQuery#am and jQuery#addAm. The hooks are all named after the unprefixed AMCSS attributes. If you are ever in doubt, the easiest way to get the hook name is $.normaliseAm(ATTRIBUTE).slice($.AM_PREFIX.length).

Type:
  • Object.<AMCSS_hook>
Example
// Suppose that an AMCSS attribute cannot have values, only the attribute
// itself.
$.amHooks.empty = {
    set: function (element, value, name) {
        element.setAttribute(name, "");
    },
    get: function (element) {
        return "";
    }
};

(static) normaliseAm(attribute) → {String}

Source:
  • jquery.amcss.js, line 411
See:
  • external:jQuery.AM_PREFIX

Normalises the attribute so that it always starts with the AMCSS prefix ("am-" by default). If the attribute already starts with the AMCSS prefix, no action is taken.

The other AMCSS methods (jQuery#am, jQuery#addAm, jQuery#hasAm and jQuery#removeAm) always pass their given attributes through this function so there is no need to do that manually.

This function is aliased as jQuery.normalizeAm.

Examples

Attributes are normalised

// $.AM_PREFIX = "am-"
$.normaliseAm("button");    // -> "am-button"
$.normaliseAm("am-button"); // -> "am-button"

Changing $.AM_PREFIX

// $.AM_PREFIX = "am-"
$.normaliseAm("button"); // -> "am-button"
$.AM_PREFIX = "data-"
$.normaliseAm("button"); // -> "data-button"

addAm(attr, value) → {jQuery}

Source:
  • jquery.amcss.js, line 749

Adds one or more values to the specified AMCSS attribute. The values can be added using a pair of strings, an object of key/value pairs or the value can be a function to derive the value. In all cases, the attribute is passed through jQuery.normaliseAm so there is no need to do this manually (although it will not cause any problems if you do). If the attribute does not exist, it is created before the value(s) is/are added.

Examples

Adding a value with a string

// Markup is <div id="one">
$("#one").addAm("alpha", "one");
// Now markup is <div id="one" am-alpha="one">
$("#one").addAm("alpha", "two three");
// Now markup is <div id="one" am-alpha="one two three">

Adding a value with a function

// Markup is <div id="one" am-alpha="one">
$("#one").addAm("alpha", function (index, value) {
    return value.toUpperCase() + "_" + index;
});
// Now markup is <div id="one" am-alpha="one ONE_0">

Adding a value with an object

// Markup is <div id="one" am-alpha="one">
$("#one").addAm({
    alpha: function (index, value) {
        return value.toUpperCase() + "_" + index;
    },
    bravo: "two three"
});
// Now markup is
// <div id="one" am-alpha="one ONE_0" am-bravo="two three">
Parameters:
Name Type Description
attr Object | String

Attribute to modify.
value function | String

Either the value(s) to add or a function that generates the value(s) to add.
Returns:

The jQuery instance.

am(attropt, valueopt) → {jQuery|Object|String|undefined}

Source:
  • jquery.amcss.js, line 621

Sets or gets an AMCSS attribute.

This function has 3 modes:
- Pass 0 arguments to get a collection of all AMCSS attributes.
- Pass 1 argument to get the value of that attribute.
- Pass 2 arguments to set the value of that attribute. In all cases, the attribute is automatically passed through jQuery.normaliseAm so there is no need to do this manually (although it will not cause any problems if you do).

Examples

Setting an attribute

// Markup is <div id="one">
$("#one").am("alpha", "one");
// Markup is now <div id="one" am-alpha="one">
$("#one").am("alpha", function (index, value) {
    return index + value + value;
});
// Markup is now <div id="one" am-alpha="0oneone">
$("#one").am({
    bravo: "two",
    charlie: "three"
});
// Markup is now
// <div id="one" am-alpha="0oneone" am-bravo="two" am-charlie="three">

Getting an attribute

// Markup is <div id="one" am-test="one">
$("#one").am("test"); // -> "one"
$("#one").am("fake"); // -> undefined

Getting all AMCSS attributes

// Markup is <div id="one" am-test="one">
$("#one").am(); // -> {"am-test": "one"}

Either the jQuery instance (when getting or setting) or an object of all the AMCSS attributes.

hasAm(attribute, valueopt) → {Boolean}

Source:
  • jquery.amcss.js, line 675

Checks to see if the AMCSS attribute exists or that it contains a specific value. Thw attribute is automatically passed through jQuery.normaliseAm so there is no need to do this manually (although it will not cause any problems if you do).

Examples

Checking for the attribute

// Markup is <div id="one" am-alpha="one two three">
$("#one").hasAm("alpha"); // -> true
$("#one").hasAm("bravo"); // -> false

Checking for the value

// Markup is <div id="one" am-alpha="one two three">
$("#one").hasAm("alpha", "one");  // -> true
$("#one").hasAm("alpha", "four"); // -> false

removeAm(attropt, valueopt) → {jQuery}

Source:
  • jquery.amcss.js, line 817

Removes values from the attribute or the attribute itself. If no arguments are passed then all AMCSS attributes are removed.

Examples

Removing value with a string

// Markup is <div id="one" am-test="one two three">
$("#one").removeAm("test", "two");
// Now markup is <div id="one" am-test="one three">

Removing an attribute with a function

// Markup is <div id="one" am-test="a0 a1 a2">
$("#one").removeAm("test", function (index, value) {
    return "a" + index;
});
// Now markup is <div id="one" am=test="a1 a2">

Removing the attribute

// Markup is <div id="one" am-test="one two three">
$("#one").removeAm("test");
// Now markup is <div id="one">

Removing all AMCSS attributes

// Markup is <div id="one" am-alpha="one" am-bravo="two">
$("#one").removeAm();
// Now markup is <div id="one">
