(private) setAttribute(element, attribute, value, index)

Source:
  • jquery.amcss.js, line 254

Sets the value of the given attribute on the given element. The attribute name is normalised (see $.normaliseAm), the set hook is executed (see $.amHooks) if it exists and if the value is a function then that function is executed before the returning value is assigned to the attribute.

Examples

Setting a property

// Markup is:
// <div id="one"></div>

var element = document.getElementById("one");
setAttribute(element, "label", "test");

// Now markup is:
// <div id="one" aria-label="test"></div>

Setting a property using a function

// Markup is:
// <div id="one" aria-label="test"></div>

var element = document.getElementById("one");
setAttribute(element, "label", function (i, attr) {
    return this.id + "__" + i + "__" + attr;
}, 0);

// Now markup is:
// <div id="one" aria-label="one__0__test"></div>
Parameters:
Name Type Description
element Element

Element whose attribute value should be set.
attribute String

Name of the attribute to set.
value function | String

Value of the attribute to set or the function that will create the attribute to set.
index Number

Index of the element within its collection - passed to value if value is a function.

A hook for an AMCSS attribute. Every property is optional so there is no need to specify one to execute the default functionality.

Be aware that these hooks only affect the AMCSS methods; jQuery#attr and jQuery#prop will not be affected by any changes here. There are similar jQuery.attrHooks and jQuery.propHooks (for set and get) that work in the same way if you need to completely control attribute/property setting.

AMCSS_hook_get(element, attribute) → {Boolean|Number|String}

Source:
  • jquery.amcss.js, line 542

Handles the getting of an AMCSS attribute. The function takes the element and should return the value that the jQuery AMCSS methods should return.

When getting an attribute, please do not use jQuery#am as this can create an infinite loop.

Example

A "loud" attribute should always have uppercase values

$.amHooks.loud = {
    get: function (element, attribute) {
        return element.hasAttribute(attribute)
            ? element.getAttribute(attribute).toUpperCase()
            : "";
    }
};

// Markup is
// <div id="one" am-loud="yes"></div>

$("#one").am("loud"); // -> "YES"

AMCSS_hook_set(element, value, attribute) → {?}

Source:
  • jquery.amcss.js, line 505

Handles the setting of an AMCSS attribute. If the function returns a value, that value is used to set the attribute; returning null, undefined, or not returning anything will prevent the normal attribute setting process from completing.

When setting an attribute, please do not use jQuery#am or jQuery#addAm as this can create an infinite loop.

Example

A "loud" attribute should always have uppercase values

$.amHooks.loud = {
    set: function (element, value) {
        return value.toUpperCase();
    }
};

// Markup is
// <div id="one"></div>

$("#one").am("loud", "yes");

// Now markup is
// <div id="one" am-loud="YES"></div>
