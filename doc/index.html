jquery.amcss.js

This is a jQuery plugin that makes it easier to work with AMCSS. Its methods include:
jQuery#am for setting an getting AMCSS attributes.
jQuery#addAm for adding values to AMCSS attributes.
jQuery#removeAm for removing values from AMCSS attributes
jQuery#hasAm for checking the existence of AMCSS attributes or values.

There is also jQuery.normaliseAm (and its alias jQuery#normalizeAm) for making sure the attribute is prefixed and jQuery#AM_PREFIX for defining that prefix. The files can be downloaded on GitHub.
